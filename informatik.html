Javascript og DOM
DOM er en slags Javascript, specifict til browser filer.
Javascript gør hjemmesiden mere dynamisk, end man kan gøre med HTML og CSS.
F.eks. når man trykker på en af sektionerne i hjemmesiden, så sker der noget, det bliver dynamisk.
Javascript gør også at man ikke behøver ændre information hvert sted det står, men man kan lave en overordnet data ændring.
